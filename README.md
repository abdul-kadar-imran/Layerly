# 📖 About

Layerly is an interactive learning platform designed to make CSS easier to understand through experimentation.

Instead of reading documentation or memorizing properties, users can directly interact with CSS concepts and immediately observe the visual results.

Whether you're learning CSS for the first time or revising advanced layout techniques, Layerly provides a practical, engaging, and intuitive learning experience.

# ⚙️ Running Locally

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Create a production build

```bash
npm run build
```

Preview the production build

```bash
npm run preview
```
